JS Games

Here is a compendium of a few games I've written in JS. I've always wanted to build games, although I've never had the opportunity to do so in a professional capacity. So around the end of November of 2019, I decided I'd sit down and figure out how to do it.

I learn the most by doing, and I thought exploring several different games in a multitude of platforms would be the best way to really gain some useful knowledge about how to produce games. The platforms I chose were ES5 JavaScript, ES6 JavaScript, and Typescript, with no frameworks or runtime-dependencies. In some cases I used examples to demonstrate broad concepts, but every line of code was written by me. This helps to exemplify my ability to problem solve (because I'm not using other folks solutions), and to organize and write code (because I'm not using other folks libraries or frameworks). None of these examples really serve a practical purpose, other than continuing education for myself, so...I guess try to keep that context in mind.
